# cgj-engine
C++ OpenGL engine   
## What to install in linux

```bash
$ sudo apt install libglew-dev mesa-utils libsfml-dev cmake
$ sudo apt-get install libgtest-dev
```

## Generate Makefiles with cmake:

```bash
$ cd Release
$ mkdir -p bin
$ cd bin
$ cmake -DCMAKE_BUILD_TYPE=Release ../
```

## **Compiling**

```bash
$ make
make
```

## **Running**

```bash
$ ./game
```
